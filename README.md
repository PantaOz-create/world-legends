# PANTA’S WORLD LEGEND NAVIGATOR v16

An interactive, single‑file web map exploring **900 myths & legends** from around the world.  
Browse by category, search by name/culture/keywords, and click map markers to open full story details plus curated “learn more” links.

## Features
- **Interactive world map** (D3.js) with selectable legend points
- **900 entries** with: name, coordinates, type, culture/origin, era, description, and tags
- **Search** across names, cultures, and descriptions
- **Category filters**: Lost Cities, Mythical Places, Underworld, Sacred Sites, and Custom
- **Connection networks** (toggleable “ley-line” style links) to visualize thematic groupings
- **Auto‑zoom** and keyboard navigation for fast exploration
- **Export / Import** (JSON) for backing up and sharing custom additions
- **Add New Legend** modal for creating your own entries

## How to use
1. Open the app in your browser.
2. Click any marker (or an item in the left list) to view details.
3. Use the filter chips and the connection network toggles to explore patterns.
4. Use **Export** to save your custom legends, and **Import** to reload them later.
