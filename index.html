How to Add Legends

1. Click + Add
Fill in the form with your legend's details.

2. Required Fields
• Name - legend name
• Latitude/Longitude - find on map (negative = West/South)
• Type - Mythical Place, Lost City, Sacred Site, or Underworld
• Culture - e.g., Mexican, Celtic, Chinese
• Description - full story

3. Tags
Comma-separated: dragon,treasure,hero

4. Share
Click Export to save your additions. Send the file to the database owner to include in main version.

5. Import
Click Import to load legends from a JSON file.

Map Tips
• Click any dot for details
• Use filter buttons to show specific types
• Toggle connection networks to see related legends
• Arrow keys to navigate, R for random
